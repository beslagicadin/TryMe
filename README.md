# TryMe

A simple Android app that changes the background to a random color every time the button is tapped.

## Features

- Single button interaction
- Random background color on each tap
- Smooth color transitions

## Tech Stack

- Java
- Android SDK
- XML layouts

## How It Works

Each tap generates a random RGB value using `Random` and applies it as the background color of the root layout via `setBackgroundColor()`.

## Getting Started

1. Clone the repository
2. Open in Android Studio
3. Build and run on an emulator or physical device

## Requirements

- Android Studio
- Android SDK
